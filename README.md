# Flashcard Quiz

A SwiftUI flashcard app for vocabulary learning with flip animations and AI-powered content generation.

## Features

- **Two View Modes**: Swipe through cards with flip animations or browse a scrollable list
- **AI Generation**: Auto-generate definitions and word types using on-device Apple Intelligence
- **Manual Selection**: Choose word types from a dropdown picker
- **Word Type Badges**: Color-coded badges for noun, verb, adjective, and more
- **Swipe Actions**: Quick edit and delete from list view
- **Persistence**: Cards saved locally with SwiftData

## Requirements

- iOS 26+
- Device with Apple Intelligence support

## Tech Stack

- SwiftUI
- SwiftData
- FoundationModels

## Installation

1. Clone the repository
2. Open `flashcard-quiz.xcodeproj` in Xcode
3. Build and run on a supported device

## Usage

**Cards Tab**
- Tap card to flip between word and definition
- Use arrow buttons to navigate
- Add, edit, or delete cards with bottom buttons

**List Tab**
- Tap any card to view in detail
- Swipe left to edit
- Swipe right to delete

**Adding Cards**
- Enter a word manually
- Select word type from dropdown or use AI Generate
- AI fills both definition and word type automatically
